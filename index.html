Assignment 1 Overview

Explore the steps and tools involved in setting up your DevOps environment. Below are the main tasks from Assignment 1. Follow these steps to get started with your journey into the world of DevOps.

Step 1: Create GitHub Account and Generate SSH Keys

Secure your GitHub connections by generating and adding SSH keys. Learn more.

Step 2: Install Homebrew

Install Homebrew to simplify package installations on your macOS or Linux system. Learn more.

Step 3: Install Terraform

Terraform is essential for managing infrastructure as code. Follow the guide to install Terraform. Learn more. brew install tfswitch

Step 4: Install AWS CLI

Set up AWS CLI to interact with AWS services directly from your terminal. Learn more. brew install awscli

Step 5: Set Up AWS Keys

Configure AWS keys in the AWS CLI to authenticate and manage your resources.

aws configure

Step 6: Create GitHub Repository

Create a new repository and push your initial commit with a README file.

git clone
git add .
git commit -m "Initial commit"
git push origin main
